# GenTok
B2C 유전자 검사 웹/앱

## Install

##### USING YARN (Recommend)
```
  1. yarn install
  2. yarn start
```
##### USING NPM
```
  1. npm i OR npm i --legacy-peer-deps
  2. npm start
```

## Story

- 유전체 검사를 위한 DTC 플랫폼 개발
- QA 테스트, UI 이슈 수정 작업
- 외주 업체 산출물 피드백 및 검수
